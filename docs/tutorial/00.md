# Overview

First time your run project you should see below output

**Command**

```shell
python -m scli --help
```

```text

  about                  Shows information about scli

```

You can get help for one of available commands;

**Command**

```shell
python -m scli check --help
# or
python -m scli help check
```

**Output**

